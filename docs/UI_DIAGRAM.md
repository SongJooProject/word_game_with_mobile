## 버튼 목록

| 버튼 | 위치 | 기능 | 표시 조건 |
|------|------|------|----------|
| 🏠 홈 | 상단 왼쪽 | 메뉴로 돌아가기 | 게임 중 |
| ☰ 메뉴 | 상단 오른쪽 | 사이드바 토글 | 항상 |
| 🔵 선택형 | 중앙 | 선택형 문제만 풀기 | 유형 선택 시 |
| ✏️ 빈칸형 | 중앙 | 빈칸형 문제만 풀기 | 유형 선택 시 |
| 🎯 전체 | 중앙 | 모든 문제 풀기 | 유형 선택 시 |
| 🚀 게임 시작 | 하단 | 게임 시작 | 유형 선택 시 |
| ← 메뉴로 돌아가기 | 하단 | 메뉴로 돌아가기 | 유형 선택 시 |
| ✅ 정답 확인 | 하단 | 정답 제출 | 문제 풀이 중 |
| 다음 문제 → | 하단 | 다음 문제로 이동 | 정답 확인 후 |
| 🔄 다시 하기 | 결과 화면 | 게임 재시작 | 게임 종료 후 |

```
                    ┌─────────────┐
                    │  과목 선택   │
                    └──────┬──────┘
                           │
                           ▼
                    ┌─────────────┐
                    │ 챕터/섹션    │
                    │   선택      │

                           │
                           ▼
                    ┌─────────────┐
                    │ 게임 유형    │
                    │   선택      │
                    └──────┬──────┘
                           │
              ┌────────────┼────────────┐
              ▼            ▼            ▼
        ┌─────────┐  ┌─────────┐  ┌─────────┐
        │ 선택형만 │  │ 빈칸형만 │  │  전체   │
        └────┬────┘  └────┬────┘  └────┬────┘
             │            │            │
             └────────────┼────────────┘
                          ▼
                   ┌─────────────┐
                   │  게임 시작  │
                   └──────┬──────┘
                          │
                          ▼
                   ┌─────────────┐
                   │  문제 풀이  │ ◄── 🏠 홈 버튼 (메뉴로 돌아가기)

                   ┌─────────────┐
```
