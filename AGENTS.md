## 셀프테스트 프로세스

### 자동 테스트 실행
모든 결정/변경 후 자동으로 테스트가 실행됩니다.

```bash
python scripts/auto_test.py
```

### 테스트 항목
1. **파일 구조 검증**: 필수 파일 존재 여부
2. **HTML 구조 검증**: 문법, 필수 태그
3. **CSS 문법 검증**: 중괄호 매칭, 기본 문법
4. **JS 문법 검증**: 괄호 매칭, 함수 정의
5. **JSON 유효성 검증**: 파싱 가능, 주제 존재
6. **Git 상태 검증**: 변경사항 확인
7. **DB 스키마 검증**: 테이블 구조

### 테스트 결과
- PASS: 정상
- FAIL: 수정 필요

### 자동화 규칙
1. 파일 변경 시 자동 테스트 실행
2. 테스트 실패 시 변경사항 롤백
3. 모든 테스트 통과 후 커밋 허용
