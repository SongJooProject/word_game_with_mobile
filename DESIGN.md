# DESIGN.MD - UI/UX 디자인 가이드라인

## 개요
법률 용어 학습 게임의 UI/UX 디자인 규칙 및 레이아웃 정의

---

## 레이아웃 구조

### 메인 레이아웃 (한 페이지 메뉴)
```
+------------------+--------------------------------------------+
|                  |  🏠  과목 이름              ⭐ 점수  📝 문제 |
|   사이드바       |--------------------------------------------|
|   (과목 선택)    |  [====================================]     |
|                  |           진행률 바 (Progress Bar)           |
|                  |                                            |
|                  |  +--------------------------------------+ |
|                  |  | 챕터 1                               | |
|                  |  | [전체] [섹션1] [섹션2] [섹션3]       | |
|                  |  +--------------------------------------+ |
|                  |  +--------------------------------------+ |
|                  |  | 챕터 2                               | |
|                  |  | [전체] [섹션1] [섹션2]               | |
|                  |  +--------------------------------------+ |
|                  |                                            |
+------------------+--------------------------------------------+
```

### 화면 전환 순서
1. **과목 선택** → 사이드바에서 과목 클릭
2. **챕터 선택** → "과목을 선택하세요" → 챕터 목록
3. **섹션 선택** → 챕터 이름 → 섹션 목록 (전체 포함)
4. **게임 유형 선택** → 선택형/빈칸형/전체
5. **게임 진행** → 문제 표시 → 정답 확인 → 다음 문제
6. **결과** → 최종 점수 표시

## 버튼 시스템

### 상단 버튼
| 버튼 | 위치 | 기능 | 표시 조건 |
|------|------|------|----------|
| 🏠 홈 | 상단 왼쪽 | 메뉴로 돌아가기 | 게임 중 |
| ☰ 메뉴 | 상단 오른쪽 | 사이드바 토글 | 항상 |

### 게임 유형 선택 버튼
| 버튼 | 기능 | 스타일 |
|------|------|--------|
| 🔵 선택형 | 선택형 문제만 풀기 | 기본 |
| ✏️ 빈칸형 | 빈칸형 문제만 풀기 | 기본 |
| 🎯 전체 | 모든 문제 풀기 | 기본 |
| 🚀 게임 시작 | 게임 시작 | 블루 |
| ← 메뉴로 돌아가기 | 메뉴로 돌아가기 | 그레이 |

### 문제 풀이 버튼
| 버튼 | 기능 | 표시 조건 |
|------|------|----------|
| ✅ 정답 확인 | 정답 제출 | 문제 풀이 중 |
| 다음 문제 → | 다음 문제로 이동 | 정답 확인 후 |
| 🔄 다시 하기 | 게임 재시작 | 결과 화면 |

## 메뉴 시스템

### 사이드바
- **위치**: 왼쪽 고정 (240px)
- **배경**: `#f7f6f3` (라이트 그레이)
- **테두리**: `#e9e9e7` (미디엄 그레이)
- **과목 목록**: 세로 스택 배치

### 메인 메뉴 영역 (한 페이지)
- **레이아웃**: 모든 챕터와 섹션이 한 페이지에 표시
- **챕터 블록**: 각 챕터마다 카드 형태로 표시
- **섹션 버튼**: 가로 배치 (flex-wrap)
- **전체 선택**: 초록색 버튼으로 강조

### 메뉴 구조
```
형사소송법
├─ 📋 관할 및 법원의 결정, 판결
│  ├─ [전체] (28문제)
│  ├─ [사물관할]
│  ├─ [토지관할]
│  ├─ [이송 및 관할이전의 신청]
│  ├─ [관할위반 및 이중기소 처리]
│  ├─ [사물관할 다른 경우(토지관할 무관)]
│  └─ [관련사건 병합관할]
└─ 📑 제척, 기피
   ├─ [전체] (26문제)
   ├─ [제척사유]
   └─ [기피사유]
```

## 문제 표시

### 선택형 (Type 1)
```
+--------------------------------------------------+
| 1번                                    🔵 선택형   |
+--------------------------------------------------+
|                                                  |
|  피해자와 사실혼 관계에 있는 법관은?              |
|                                                  |
+--------------------------------------------------+
|  1. 친족 또는 친족관계에 있었던 자(2호)           |
|  2. 사건에 관하여 증인, 감정인이 된 때(4호)       |
|  3. 사건에 관하여 수사기관 직무를 행한 때(6호)     |
|  4. 전심재판 관여(제7호)                          |
|  5. 전심재판 기초되는 조사, 심리에 관여(7호)      |
|  6. 제척사유 아님                                |
+--------------------------------------------------+
|          ✅ 정답 확인                             |
+--------------------------------------------------+
```

+--------------------------------------------------+

+--------------------------------------------------+

|                                                  |

## 애니메이션

### 전환 효과
- **슬라이드 인**: `0.3s ease` (문제 카드)
- **페이드 인**: `0.2s ease` (메시지)

### 인터랙션
- **호버**: 배경색 변화 (0.15s ease)
- **포커스**: 블루 테두리 + 그림자

---

## 접근성

### 키보드 단축키
- **Enter**: 다음 동작 (정답 확인 또는 다음 문제)
- **Escape**: 뒤로 가기 또는 사이드바 토글

### 시각적 피드백
- **정답**: 초록색 테두리 + 배경
- **오답**: 빨간색 테두리 + 배경 + 쉐이크
- **호버**: 색상 변화 + 약간의 이동

## 데이터 구조

### 문제 JSON 형식
```json
{
  "subjects": [
    {
      "name": "형사소송법",
      "chapters": [
        {
          "name": "관할 및 법원의 결정, 판결",
          "questions": [
            {
              "type": "type1",
              "section": "사물관할",
              "question": "문제 텍스트",
              "options": ["보기1", "보기2", "보기3"],
              "answer": 1
            },
            {
              "type": "type2",
              "section": "토지관할",
              "question": "토지관할은 <범죄지>, 피고인의 주소, 거소 또는 <현재지>로 한다.",
              "answer": "범죄지|현재지"
            }
          ]
        }
      ]
    }
  ]
}
```

### 빈칸 형식
- **구분자**: `<빈칸텍스트>` (.angle brackets)
- **정답 구분**: `|` (파이프)
- **예시**: `<범죄지>|<현재지>` → 정답: `범죄지|현재지`
