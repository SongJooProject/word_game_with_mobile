# DESIGN.MD - word-game 디자인 가이드라인

## 개요
법률 용어 빈칸 채우기 게임의 디자인 사양을 관리합니다.

## 색상 체계
| 항목 | 색상 | HEX |
|------|------|-----|
| 배경 | 다크 네이비 | #1a1a2e |
| 사이드바 | 딥 블루 | #16213e |
| 포인트 | 코랄 레드 | #e94560 |
| 서브 포인트 | 퍼플 | #533483 |
| 텍스트 | 화이트 | #ffffff |
| 정답 | 민트 | #4ecca3 |
| 오답 | 레드 | #e94560 |

## 레이아웃
```
+------------------+--------------------------------+
|                  |                                |
|    사이드바      |           메인 영역            |
|    (200px)       |                                |
|                  |   헤더 (제목 + 점수)           |
|  - 주제 리스트   |                                |
|                  |   게임 영역                     |
|                  |   - 문장 (빈칸 포함)            |
|                  |   - 힌트                       |
|                  |   - 입력 필드                   |
|                  |   - 버튼                       |
|                  |                                |
+------------------+--------------------------------+
```

## 컴포넌트 사양

### 사이드바
- 너비: 200px
- 배경: #16213e
- 항목: 패딩 15px, 마진-bottom 10px
- 호버: 배경 #e94560, translateX(5px)
- 활성 상태: 배경 #e94560, 굵은 글씨

### 메인 영역
- 패딩: 30px
- 헤더: flex, space-between
- 점수판: gap 30px

### 게임 영역
- 문장: 폰트 1.5em, 패딩 30px, 배경 #16213e, 둥근 모서리 15px
- 입력 필드: 패딩 15px 25px, 폰트 1.2em, 너비 180px
- 버튼: 패딩 15px 35px, 둥근 모서리 10px

### 애니메이션
- 트랜지션: 0.3s
- 호버 효과: translateY(-3px), box-shadow

## 폰트
- 메인: 'Malgun Gothic', sans-serif
- 제목: 2em, 굵은 글씨
- 본문: 1.5em
- 입력: 1.2em
